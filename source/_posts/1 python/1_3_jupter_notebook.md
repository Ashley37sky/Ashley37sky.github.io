# 快捷键

+ `h` 查看所有快捷键

<!--more-->

+ + `Enter` 从命令模式进入编辑模式
  + `Esc`    从编辑模式退回到命令模式
+ + `m` 将代码单元格转换为Markdown单元格
  + `y` 将Markdown单元格转换为代码单元格
+ + `shift+Enter` 运行本单元格，选择下面的代码块
  + `ctrl+Enter` 运行本单元格
  + `alt+Enter` 运行本单元格，在下方新建一个单元格
+ + `a` 在上方新建一个单元格（above）
  + `b` 在下方新建一个单元格（below）
+ + `d` 删除选中的单元格（delete）
  + `z` 恢复
  + `x` 剪切本单元格
  + `c` 复制本单元格
    + `shift v` 粘贴到上面
    + `v` 粘贴到下面
+ `l` 显示代码行号
